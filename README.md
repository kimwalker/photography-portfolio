# Photography Portfolio

Built with Astro + Decap CMS, hosted on Cloudflare Pages.

## Setup

1. Push this repo to GitHub
2. Connect to Cloudflare Pages
   - Build command: `npm run build`
   - Output directory: `dist`
3. Enable Netlify Identity on a Netlify project
   - Add your Cloudflare Pages URL as an allowed redirect URL
   - Enable Git Gateway

## Local development

```bash
npm install
npm run dev
```

## CMS

Access the CMS at `/admin` once deployed and Netlify Identity is configured.
